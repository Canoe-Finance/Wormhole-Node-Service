# Canoe cross-chain black-end base on wormhole

## Description

### How to use

```bash
{
 "userPublicKey": "[wallet publicKey]",
 "mint": "[token address]",
 "targetAddress": "[eth address]",
 "amount": "[cross-chain amount]"
}
```

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
